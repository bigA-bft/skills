## 创建项目

1. 创建时需要了解清楚项目需求, 有什么不懂的及时提出
2. 需要集成uni-ui
3. 需要支持多国语言,简体中文,英文,繁体中文, 俄语, 德语
4. 仅需支持安卓和ios平台
5. 需要支持夜间模式

## 开发uniapp页面

1. 使用vue3版本开发
2. 优先内置组件, 再考虑使用uni-ui组件库
3. 开发业务时需先了解清楚文档再开发
4. 开发完成需要先测试


## 参考文档

- https://uniapp.dcloud.net.cn/quickstart.html

## 开发规范

1. 使用vue3版本开发
2. css 使用BEM命名规范

## 风格
- 尽可能简洁
- 使用卡片设计, 圆角12px
