# 计算器技能

## 功能

- 支持加减乘除四则运算
- 显示详细计算步骤
- 支持整数和小数
- 自动遵循运算优先级（先乘除后加减）

## 使用方法

当用户提出计算需求时，调用 Python 脚本执行计算：

```bash
python .claude/skills/calculator/scripts/calculate.py "表达式"
```

## 支持的表达式格式

- 基本运算：`10 + 20`，`100 - 50`，`5 * 8`，`20 / 4`
- 复合运算：`10 + 20 * 3 - 5 / 2`
- 小数运算：`3.14 * 2.5`，`10.5 + 5.25`
- 负数运算：`-5 + 10`，`20 * -3`

## 示例

### 示例 1：基本加法
用户："帮我算一下 123 + 456"
```bash
python .claude/skills/calculator/scripts/calculate.py "123 + 456"
```
输出：
```
计算表达式：123 + 456

123 + 456 = 579

最终结果：579
```

### 示例 2：混合运算
用户："计算 10 + 20 * 3"
```bash
python .claude/skills/calculator/scripts/calculate.py "10 + 20 * 3"
```
输出：
```
计算表达式：10 + 20 * 3

20 × 3 = 60
10 + 60 = 70

最终结果：70
```

### 示例 3：除法
用户："100 除以 3 等于多少"
```bash
python .claude/skills/calculator/scripts/calculate.py "100 / 3"
```
输出：
```
计算表达式：100 / 3

100 ÷ 3 = 33.33333333333333333333333333

最终结果：33.33333333333333333333333333
```

## 注意事项

1. 运算优先级：自动先计算乘除，再计算加减
2. 除零保护：除数为零时会返回错误提示
3. 数字格式：支持整数和小数
4. 表达式中可以不包含空格，如 `10+20*3` 和 `10 + 20 * 3` 结果相同
